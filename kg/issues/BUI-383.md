## 목표

`/built:iter`와 `/built:report` standalone skill 문서가 설치형 plugin 환경에서도 target project cwd를 유지하고, helper script는 plugin root 기준으로 찾도록 정렬한다.

## 구현 내용

- `skills/iter/SKILL.md`와 `skills/report/SKILL.md`의 standalone 실행 예시를 `BUILT_PLUGIN_DIR` 기반 `SCRIPT_DIR` 호출로 정리했다.
- target project root와 plugin root가 분리된다는 실행 계약을 두 skill 문서에 명시했다.
- `test/plugin-packaging.test.js`가 Plan/Design/Run뿐 아니라 Iter/Report skill의 cwd 상대 `node scripts/...` 호출, `BUILT_PLUGIN_DIR` 누락, `BASH_SOURCE` 기반 경로 추정 회귀를 잡도록 확장했다.
